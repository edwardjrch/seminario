OBJETIVOS

Como resultado del desarrollo de las actividades de la clase, se espera que el estudiante esté en capacidad de realizar rutinas en Python para:

  • Integrar, limpiar, tratar y estructurar datos provenientes de diferentes fuentes de información.
  • Generar reportes interactivos y recursos visuales a partir de diferentes bases de datos.
  • Crear simulaciones y modelos matemáticos estocásticos para hacer inferencia.
